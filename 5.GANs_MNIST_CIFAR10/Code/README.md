1. *FullyConnectedMNIST* \
Contains implementations for the generation of novel data based on the **MNIST** dataset. The discriminator and generator networks are based on fully connected layer solely.
2. *CNN_MNIST* \
Contains implementations for the generation of novel data based on the **MNIST** dataset. The discriminator and generator networks are based on CNNs and densely connected layers.
3. *CNN_CIFAR10* \
Contains implementations for the generation of novel data based on the **CIFAR10** dataset. The discriminator and generator networks are based on CNNs and densely connected layers.
